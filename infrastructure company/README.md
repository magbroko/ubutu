# Energy Infrastructure — Corporate Website

A premium corporate website for an integrated energy infrastructure and oilfield production systems company operating across the UK and West Africa.

## Overview

- **Positioning**: Strategic infrastructure partner, disciplined operator, long-term asset builder
- **Design**: Minimal, institutional, dark/light themes, navy/graphite/steel with gold accents
- **Tone**: Credible, strategic, globally competitive

## Quick Start

1. Open `index.html` in a browser, or
2. Serve locally: `npx serve .` or `python -m http.server 8000`

## Structure

```
├── index.html      # Main single-page site
├── styles.css      # Premium institutional styling
├── script.js       # Interactions, theme toggle, metrics, form
├── assets/         # Images, video (see assets/README.md)
└── README.md
```

## Customisation

- **Contact details**: Update email, phone numbers in `index.html` (Contact section)
- **Metrics**: Adjust values in `index.html` (data-target attributes) and labels
- **Analytics**: Add your GA4 or other analytics script in `script.js` (see comment block)
